# Словник із флеш-картками

Консольна утиліта для вивчення іноземних слів методом флеш-карток з алгоритмом інтервального повторення.  
Навчальна практика, варіант Б-13.

## Запуск

```bash
python main.py
```

Python 3.10+. Зовнішні бібліотеки не потрібні.

## Функції

| # | Функція | Опис |
|---|---------|------|
| 1 | Додати слово | Слово + переклад + приклад, перевірка дублікатів |
| 2 | Переглянути словник | Таблиця всіх карток з рейтингами |
| 3 | Тренування: слово → переклад | Інтервальне повторення |
| 4 | Тренування: переклад → слово | Зворотній режим — складніший рівень |
| 5 | Статистика | Топ-5 кращих і гірших слів, середній рейтинг |
| 6 | Пошук слова | Пошук за підрядком у слові або перекладі |
| 7 | Редагувати картку | Змінити переклад або приклад без видалення |
| 8 | Скинути рейтинг | Одне слово або весь словник — вчити заново |
| 9 | Імпорт із файлу | Завантажити список слів із `.txt` файлу |
| 10 | Видалити слово | Видалення за ID або назвою |

## Алгоритм інтервального повторення

Кожне слово отримує вагу: `вага = max(1, 5 − рейтинг)`.  
Нові слова (рейтинг 0) з'являються у **5 разів частіше** за добре засвоєні (рейтинг ≥ 4).  
Реалізовано через `random.choices()` — без зовнішніх бібліотек.

## Імпорт слів із файлу

Створіть `.txt` файл — один рядок на слово:

```
# це коментар, ігнорується
apple - яблуко - I eat an apple every day.
book - книга
cat - кіт - My cat sleeps all day.
```

Формат: `слово - переклад` або `слово - переклад - приклад`.  
Готовий приклад: [`words_example.txt`](words_example.txt)

## Формат даних

`words.json` створюється автоматично при першому запуску:

```json
[
  {
    "id": 1,
    "word": "apple",
    "translation": "яблуко",
    "example": "I eat an apple every day.",
    "rating": 3
  }
]
```
