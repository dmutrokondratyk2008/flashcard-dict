# Словник із флеш-картками

Консольна утиліта для вивчення іноземних слів методом флеш-карток.
Навчальна практика, варіант Б-13.

## Запуск

```bash
python main.py
```

Python 3.10+. Зовнішні бібліотеки не потрібні.

## Функції

| # | Функція | Опис |
|---|---------|------|
| 1 | Додати слово | Слово + переклад + приклад вживання |
| 2 | Переглянути словник | Таблиця всіх карток з рейтингами |
| 3 | Режим тренування | Інтервальне повторення з підрахунком % |
| 4 | Статистика | Топ-5 кращих і гірших слів |
| 5 | Пошук слова | Пошук за підрядком |
| 6 | Видалити слово | Видалення за ID або назвою |

## Алгоритм інтервального повторення

Вага слова для вибірки = `max(1, 5 − рейтинг)`.
Слова з рейтингом 0 з'являються у 5 разів частіше, ніж вивчені.

## Дані

Зберігаються у `words.json` (створюється автоматично при першому запуску).
